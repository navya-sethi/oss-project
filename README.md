# Open Source Software Project

## Description
This project is created as part of the Open Source Software course. it demonstrated basic Linux shell scripting using Bash. The project includes multiple scripts that perform different operations like system information, package checking, directory creation, user input handling, and file checking.

## Features
- Displays syestem information
- Checks if a package is installed
- Creates multiple directories using loop
- Takes user input and checks number type
- Verifies if a file exists

## Requirements
- Ubuntu Linux
- Bash Shell
- Git

##How to Run

1. Open terminal
2. Navigate to project folder:
   cd oss-project

3. Give permission:
   chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

4. Run scripts:
   ./script1.sh
   ./script2.sh
   ./script3.sh
   ./script4.sh
   ./script5.sh

## Scripts Explanation

- script1.sh: Displays system information
- script2.sh: Checks if Git package is installed
- script3.sh: Creates directories using loop
- script4.sh: Takes user input and checks positive/negative
- script5.sh: Checks if file exists
